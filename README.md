1. clone app from git hub
2. install node_modules with
```bash
  npm install 
```
3. run docker to load postgreSQL
```bash
  docker-composer up -d
```
4. Start the app with 

```bash
  npm run start:dev
```
5. load seeds for basic examples

```bash
  http://localhost:3000/seeds
```
